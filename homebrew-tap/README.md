# homebrew-pingbar

Homebrew tap for [PingBar](https://github.com/jedisct1/pingbar) -- a lightweight macOS menu bar app for network connectivity monitoring and DNS management.

## Installation

```bash
brew tap jedisct1/pingbar
brew install --cask pingbar
```

## Updating

```bash
brew upgrade --cask pingbar
```

## Uninstallation

```bash
brew uninstall --cask pingbar
brew untap jedisct1/pingbar
```

## About

This tap is automatically updated when new PingBar releases are published.
